# Ethereum Classic DAO

This repository hosts a minimal static site for Ethereum Classic DAO — the off‑chain legal entity
(Ethereum Classic DAO LLC) that supports coordination, treasury administration, and operational
activities for the Ethereum Classic ecosystem.

Included pages:
- `index.html` — Home / brand hub
- `about-llc.html` — About the LLC and scope of work
- `governance-llc.html` — Governance, roles, and links to protocol work
- `treasury.html` — Treasury & grants (operational overview)
- `contact.html` — Contact and legal

Notes:
- Technical ECIP development and protocol upgrade content is maintained at the canonical ECIP
	repository: https://github.com/ethereumclassic/ECIPs
- ECIP sources present here have been archived in `ecip-archive/` for reference and may be removed
	once the site refresh is finalized.
